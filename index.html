A bit about me

Hi I'm Aman Sharma, since 2017, I've been working as a driver for Uber and Lyft. My hard work paid off when I received a scholarship from Uber, which allowed me to start my education at Arizona State University (ASU) in 2020. I just graduated last month, in May 2024, with a Bachelor of Science in Software Engineering.

During my time at ASU, I gained skills in various programming languages including Java, Python, C/C++, JavaScript (React), SQL, VBA, HTML, CSS, Bash, and MATLAB. I also became proficient in using tools like Visual Studio Code, Atom, Eclipse, IntelliJ, CLion, Android Studio, Git, Docker, Gradle, Excel, Node.js, JIRA, Sketch, and Astah. My coursework covered areas such as Algorithms and Data Structures, Database Management, Object-Oriented Programming, Distributed Software Systems, Secure Software Systems, and Web Application Development.

I had the opportunity to intern with the Fire Department, where I developed an emergency reporting app using JavaScript, React Native, Node.js, Expo, and SQLite. This mobile app enabled efficient incident reporting by allowing GPS location transmission, emergency type selection, and multimedia attachments.

Now, I am actively seeking job opportunities where I can apply my skills and contribute to impactful projects. If you are looking for a dedicated and skilled software engineer, I would love to connect and discuss how I can be a valuable addition to your team.
